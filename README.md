CSV

Простой скрипт для обработки CSV-файлов с возможностью фильтрации и агрегации данных.



Возможности

Фильтрация по одной колонке с операторами: больше (`>`), меньше (`<`), равно (`=`)
Агрегация по одной числовой колонке: среднее (`avg`), минимум (`min`), максимум (`max`)
Вывод результата в удобной табличной форме в консоль


Установка

Рекомендуется использовать виртуальное окружение.

pip install -r requirements.txt
